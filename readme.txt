=== MP3 URL ===
Tags: Links, HREF, Click, MP3, Audio, HTML5
Contributors: Liam Parker
Requires at least: 2.7
Tested up to: 3.0
Stable Tag: 1.1

== Description ==

Replaces MP3 URL's wrapped in "[" and "]" with an MP3 player, so you can listen to MP3's on your site.

NOTE: Will not work in Internet Explorer 6, 7, or 8, because the HTML5 audio tag is not supported.
